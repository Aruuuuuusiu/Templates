## All Templates

### 1. Alexandria Library

![Alexandria Preview](https://ik.imagekit.io/ky5pz0iof/Screenshot%202026-03-23%20222225.png)

**Overview**  
A modern, elegant library website template designed for showcasing collections, digital archives, or educational platforms.  
Built with a clean layout, smooth navigation, and a strong content-first approach.

**Features**
- Structured content display
- Smooth navigation
- Minimal & focused UI
- Responsive design
